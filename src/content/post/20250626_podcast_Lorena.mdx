Lorena Pantano and Alexandra Bartlett discuss a large bulk RNA-seq project that yielded lasting changes to their group’s everyday bioinformatics practices via the creation of parameterized R Markdown code templates. In the "**Quick Sip**" segment, they discuss [_reticulate_](https://rstudio.github.io/reticulate/) for managing python environments in an R context, and in "**Brewing Up Answers**", they reflect on the differences between industry and academia bioinformatics.
