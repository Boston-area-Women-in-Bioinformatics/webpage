## About

The Web Page committee maintains and develops our organization's website, ensuring it serves as an effective platform for sharing information, resources, and engaging with our community.

## Technologies

Our website is built with modern web technologies including:

- Astro for static site generation
- TypeScript for type-safe code
- Tailwind CSS for styling
- Markdown for content management
